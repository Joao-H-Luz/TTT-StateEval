## Descrição do Projeto

Este projeto tem como objetivo desenvolver um sistema de Inteligência Artificial capaz de **classificar o estado de um jogo da velha (3x3)** a partir de um tabuleiro fornecido como entrada.

A IA **não joga diretamente**, mas atua como um avaliador do estado do jogo, classificando-o em uma das seguintes categorias:

- Tem jogo  
- Possibilidade de fim de jogo  
- Empate  
- O vence  
- X vence
